# 免責事項

本リポジトリ（`income-tax-structured-json`）に含まれるデータおよびスクリプトは、情報提供を目的としたものです。

## 提供物の位置づけ

- 本リポジトリは、e-Gov法令APIから取得した法令原文と、それを構造化した計算ロジック・実務解釈データを提供します
- これらのデータは「一次情報の整理・構造化」であり、税務相談・税務判断を行うものではありません

## 3層の責任分離

┌─────────────────────────────┐
│ 利用者 最終判断と責任 │
│ ↑ │
│ AI 分析・論点整理 │
│ ↑ │
│ 本データ 一次情報の提供のみ │
└─────────────────────────────┘

## 想定される使い方

- 個人事業主・フリーランスの確定申告における参考情報として
- AIアシスタントへの知識ベースとして
- 法令学習・自習の補助資料として
- 税理士等の有資格者がクライアントに説明する際の補助資料として

## 想定されていない使い方

- **税理士法第52条**に定める税理士業務（有償の税務相談・申告代理）の代替
- 税務署への申告書類の作成を自動化すること
- 「AIが判断したので正しい」として税務調査に対応すること

## データの正確性について

- 法令は頻繁に改正されます。本リポジトリのデータは取得時点の法令を反映しており、最新の法令と異なる場合があります
- 法改正への追従はコミット履歴で確認できますが、リアルタイムの更新を保証するものではありません
- 税務判断に使用する場合は、必ずe-Gov法令検索（https://laws.e-gov.go.jp/）で最新の法令を確認してください
