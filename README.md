# income-tax-structured-json

所得税法の条文をAIが推論・計算に使える形に構造化した3層JSONデータです。

## 概要

このリポジトリは、所得税法（昭和四十年法律第三十三号）第2編第2章第4節「所得控除」の条文を、以下の3層に分離して構造化したものです。

| 層    | フォルダ     | 内容                                                                                                         |
| ----- | ------------ | ------------------------------------------------------------------------------------------------------------ |
| 第1層 | `law_text/`  | 法令原文。e-Gov法令APIから取得したJSONを**一切改変せず**格納。SHA-256ハッシュで完全性を保証                  |
| 第2層 | `law_index/` | 計算ロジック。条文から抽出した計算式・条件分岐・変数定義を7つの型に分類し構造化                              |
| 第3層 | `playbook/`  | 実務プレイブック。国税庁タックスアンサー番号、実務上の注意点（pro_tip）、よくある誤解（common_mistake）、FAQ |

## 7つの計算型

| 型  | 名称             | 計算構造                             |
| --- | ---------------- | ------------------------------------ |
| 型1 | 固定額控除       | 定額 または Σ(単価×人数)             |
| 型2 | 所得制限控除     | 所得に応じた段階テーブル参照         |
| 型4 | 寄附金控除       | min(支出額, 所得×40%) − 2,000円      |
| 型6 | 閾値控除         | max(0, 支出額 − 閾値) + 上限キャップ |
| 型7 | 段階キャップ控除 | 支払額の段階関数 + 全体上限          |

## 検証

全125テストケース合格。`scripts/validate_all.py` で検証可能。

```bash
python scripts/validate_all.py
```
